# Book Shop
<h3>📖 Description:</h3>

- A book website where users can perform CRUD operations, featuring secure user authentication with a connected server and database.


<h3>🚀 Demo:</h3>

- [Live Preview](https://book-shop-website-frontend.onrender.com)

<h3>🛠️ Technologies:</h3>

- `React.js`
- `CSS3`
- `Node.js`
- `Express.js`
- `MongoDB`


<h3>💻 Functionalities:</h3>

- Secure user authentication (Login)
- Project management (CRUD: Create, Read, Update, Delete)
- Interactive UI elements
- Dynamic content filtering by ratings

<h3>⚙️ Installation:</h3>

To run this project locally, you can follow these steps:

1. Clone the repository:
```bash
git clone https://github.com/prayer-on/book-shop-website.git
```

2. Select the Project folder:
```bash
cd book-shop-website
```

3. Select the Backend folder:
```bash
cd Backend
```

4. Install the dependencies:
```bash
npm install --legacy-peer-deps
```

5. Start the server:
```bash
npm run dev
```

```After activated the Backend, open another termianl:```

6. Select the project:
```bash
cd my-old-grimoire
```

7. Open the browser:
```bash
npm start
```
